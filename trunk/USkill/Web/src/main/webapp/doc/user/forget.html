用户找回密码
接口地址：{domain}/user/forget
请求方式：POST
接口描述：用户找回密码接口。前端需要对手机号和密码做有效验证，验证通过后方可调用该接口提交。以下几点需要注意：
1、调用“手机号”接口验证输入的手机号是否注册，如果未注册则不允许找回密码。
2、调用“生成图片”接口生成图片验证码，用户输入图片验证码后，调用“验证图片”接口验证图片验证码。
3、调用“生成短信”接口生成短信验证码，用户输入短信验证码后，调用“验证短信”接口验证短信验证码。
4、新密码长度必须大于或等于6位，密码必须MD5加密后再提交。
请求参数：
参数名必选类型说明
cellPhoneYesString11位有效手机号，前端验证为合法手机号后方可提交
passwordYesString长度大于或等于6位的密码，前端MD5加密后提交
confirmedPasswordYesString必须和password参数值一致
SMSCodeYesString短信验证码
返回示例：
{ "error_code": 0, "data": { "uid": "1", "username": "12154545", "name": "吴系挂", "groupid": 2 , "reg_time": "1436864169", "last_login_time": "0", } }
返回说明：
字段名类型说明
successboolean“true/false”代表密码重置“成功/失败”
messagestring返回消息
