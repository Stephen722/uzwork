用户登录
接口地址：{domain}/user/login
请求方式：POST
接口描述：用户登录接口。客户端需要对手机号和密码做有效验证，验证通过后方可调用该接口登录。以下几点需要注意：
1、密码长度必须大于或等于6位，密码必须MD5加密后再提交。
2、如果用户登录失败大于或等于3次，则调用“生成图片验证码”接口显示图片验证码。
3、登录成功后，后端会返回新的token，用其替换本地已保存的token。
请求参数：
参数名必选类型说明
cellPhoneYesString11位有效手机号，前端验证为合法手机号后方可提交
passwordYesString长度大于6位的密码，前端MD5加密后提交
imageCodeNoString当登录失败次数大于或等于3，需验证图片验证码
返回示例：
{ "message": "手机号或密码错误", "success": false, "value": 1 }
返回说明：
字段名类型说明
successboolean“true/false”代表“登录成功/登录失败”
messagestring返回消息
valueint登录失败次数
