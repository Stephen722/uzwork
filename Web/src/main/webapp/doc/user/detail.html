用户详情
接口地址：{domain}/user/get
请求方式：GET
接口描述：获取用户所有信息
请求参数：
参数名必选类型说明
userIdYesString加密的用户ID
返回示例：
{ "error_code": 0, "data": { "uid": "1", "username": "12154545", "name": "吴系挂", "groupid": 2 , "reg_time": "1436864169", "last_login_time": "0", } }
返回说明：
字段名类型说明
successboolean“true/false”代表获取用户信息“成功/失败”
messageString返回消息
