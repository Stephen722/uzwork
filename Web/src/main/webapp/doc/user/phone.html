验证手机号是否注册
接口地址：{domain}/user/phone
请求方式：GET
接口描述：验证手机号码是否已注册。为了防止恶意循环遍历猜测注册用户手机号，如果同一个token用不同的手机号在一天内请求该接口达10次以上，疑似恶意攻击，后端返回错误提示。
请求参数：
参数名必选类型说明
cellPhoneYesString11位有效手机号，前端验证为合法手机号后方可提交
返回示例：
{ "message": "未注册", "success": false }
返回说明：
字段名类型说明
successboolean“true/false”代表“已注册/未注册”。
如果超过访问限制，疑似恶意攻击，后端返回false
messageString返回消息“已注册”、“未注册”、“疑似恶意攻击”
