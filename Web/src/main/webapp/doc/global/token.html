生成访问Token
接口地址：{domain}/token
请求方式：GET
接口描述：在客户端开始访问之前，判断本地是否有token：
1、无token，调用该接口获取访问token，并将新生成的token和当前IP地址保存到本地供后续使用。
2、有token，判断当前IP和已经保存的IP是否一致，不一致则调用该接口生成新的token，并将新token和新IP地址保存到本地；IP没有变化，则不需要做任何事情。（该逻辑只适用APP，不适用浏览器访问）
请求参数：
参数名必选类型说明
N/AN/AN/AN/A
返回示例：
{ "message": "", "success": true, "value": "9b6ebfb1fbb2af5b0467fb7ce101940f0ee0ef1f836ada1c223a8e0fbc67fa301c0906f432f77562" }
返回说明：
字段名类型说明
successboolean“true/false”代表“成功/失败”
messageString返回消息
valueString新生成的访问token
