基础参数
说明：系统所有接口访问请求都需要在头部 http header 中包含以下两个基础参数，后续接口定义中不再对这些参数做定义描述。
请求参数：
参数名必选类型说明
Accept-ChannelYesString请求来源，分为APP和浏览器；APP访问时其值为"UZAPP_V1"，浏览器访问时其值为"browser"。
Access-TokenYesString访问token，如果无法从本地取得token或其值为空，则可以通过调用“生成token”取得，并将其保持到本地，以便后续使用。
任何访问请求如不包含以上基本参数或参数无效，被认为是一个无效请求，则返回：
返回示例：
{ "message": "Access forbidden", "success": false }
任何访问请求的正常返回结果都包含以下两个字段：
返回说明：
字段名类型说明
successboolean“true/false”代表“成功/失败”
messageString返回消息
