生成短信验证码
接口地址：{domain}/code/SMS
请求方式：GET
接口描述：生成短信验证码，并将其保存在服务器端对应token中待后续验证。短信验证码默认有效期为10分钟。
请求参数：
参数名必选类型说明
imageCodeYesString图片验证码
返回示例：
{ "error_code": 0, "data": { "uid": "1", "username": "12154545", "name": "吴系挂", "groupid": 2 , "reg_time": "1436864169", "last_login_time": "0", } }
返回说明：
字段名类型说明
successboolean“true/false”代表短信验证“成功/失败”
messagestring返回消息
